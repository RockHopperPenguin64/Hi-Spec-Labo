このブログ「ハイスペラボ」について

「ハイスペラボ」へ　ようこそ！

「ハイスペラボ」は、“楽しみながら学ぶ”をテーマにした個人ブログです。
Web制作脱初心者の自分が、手を動かしながら試行錯誤した記録やアイデアを記録していきます。
たとえ完璧でなくても、やる。その一歩を重ねることで自分のみている世界が変わると思ったのです。
このブログのおかげで、誰かが「やってみようかな」と思うきっかけになれたら嬉しいです。

このブログで書いていること

  • HTML・CSS・JavaScript を使ったブログ制作の過程やちょっとした工夫
  • 自作テーマやレイアウト、UIの試行錯誤
  • Webサイトやアプリを見て気づいた構成の分析メモ
  • 自分が感じた「気づき」や「詰まりポイント」とその乗り越え方
  • 自分もつまづた、初心者がつまずきやすい部分の記録

このサイトは、“自分のノートを誰かと共有する感覚”で書いています。
だからこそ、正直に、ていねいに、でもラフに。そんな言葉選びを心がけています。

書いている人について

このブログを書いているのは、鳥と音楽が好きな者・RockHopperPenguin64（通称：64）です。

  • Web制作に出会ってから、コードに触れるのが楽しくてしょうがない
  • 読書をするのが好き
  • 鳥が大好き。その中でもペンギンはイチオシ

このブログは、わたしのアウトプットの場であると同時に、自分が一歩踏み出す練習の場所でもあります。
このブログを読んでくださってるみなさんと、少しずつ繋がっていけたら、とてもうれしいです。

ブログの技術的な裏側

  • HTMLとCSSを使ってすべて手書きで構築
  • デザインはノート風＆落ち着いた雰囲気を意識
  • 使用ツール：VS Code / GitHub
  • サイトのホスティングは GitHub Pages
  • スマホ対応やアクセシビリティにも少しずつ取り組んでいます

ブログ自体がラボであり、私のポートフォリオでもあります。
コードも、デザインも、自分なりに工夫して育てていく感覚で向き合っています。

読んでくれるあなたへ

このブログは、「Webに興味はあるけど何から始めたらいいかわからない」と思っている人にも届いてほしいと思っています。
自分も同じだったからこそ、最初の小さな一歩を踏み出す怖さも、そこから見えるワクワクも、全部残していきたいです！

このブログが、“誰かのファーストステップ”をそっと後押しするラボになれたらうれしいです。
